# Fabflix-iOS

A simple Android version of [Fabflix](https://github.com/mingwandowski/Fabflix)


## Functionalities

 * [x] Login authorization
 * [x] Search movie based on title
 * [x] Fuzzy search
 * [x] Single movie page
